su_image_resize

Resizes an image and returns an array containing the resized URL, width, height and file type.

Description

su_image_resize( $url, (constant) $width = NULL, (constant) $height = NULL, (constant) $crop = true, (bool) $retina = false ); 

Uses native Wordpress functionality.

The url.
1. $width — Optional. (constant) => NULL
The width.

The height.
3. $crop — Optional. (constant) => true
The crop.
4. $retina — Optional. (bool) => false
