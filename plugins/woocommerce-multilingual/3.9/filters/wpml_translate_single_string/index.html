wpml_translate_single_string

The WooCommerce Multilingual wpml translate single string hook.

Description

apply_filters( 'wpml_translate_single_string', (unknown) $label, (string) $wordpress, (string) $taxonomy_singular_name_label, (unknown) $lang ); 

Parameters (4)

0. $label (unknown)
The label.
1. $wordpress (string) => 'WordPress'
The wordpress.
2. $taxonomy_singular_name_label (string) => 'taxonomy singular name: '.$label
The taxonomy singular name label.
3. $lang (unknown)
The lang.

To run the hook, copy the example below.
  1. $label = apply_filters( 'wpml_translate_single_string', $label, $wordpress, $taxonomy_singular_name_label, $lang );
