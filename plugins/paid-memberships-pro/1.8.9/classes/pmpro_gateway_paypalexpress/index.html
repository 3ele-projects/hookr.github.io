  821.      * 
  822.      * @param    string    The API method name 
  823.      * @param    string    The POST Message fields in &name=value pair format 
  824.      * @return    array    Parsed HTTP Response body

  826.     function PPHttpPost($methodName_, $nvpStr_) {
