woocommerce_payment_gateways

The WooCommerce QuickPay woocommerce payment gateways hook.

Description

add_filter( 'woocommerce_payment_gateways', (string) $add_quickpay_gateway ); 

Parameters (1)

0. $add_quickpay_gateway (string) => 'add_quickpay_gateway'
The add quickpay gateway.

Usage

To run the hook, copy the example below.
  1. $add_quickpay_gateway = apply_filters( 'woocommerce_payment_gateways', $add_quickpay_gateway ); 
  2.                          
  3. if ( !empty( $add_quickpay_gateway ) ) { 
  4.                          
  5.    // everything has led up to this point... 
  6.                          
  7. } 
  8.   
The following example is for adding a hook callback.
  1. // define the woocommerce_payment_gateways callback 
  2. function filter_woocommerce_payment_gateways( $add_quickpay_gateway ) { 
  3.     // make filter magic happen here... 
  4.     return $add_quickpay_gateway; 
  5. }; 
  6.          
  7. // add the filter 
  8. add_filter( 'woocommerce_payment_gateways', 'filter_woocommerce_payment_gateways', 10, 1 ); 
To remove a hook callback, use the example below.
  1. // remove the filter 
  2. remove_filter( 'woocommerce_payment_gateways', 'filter_woocommerce_payment_gateways', 10, 1 );
