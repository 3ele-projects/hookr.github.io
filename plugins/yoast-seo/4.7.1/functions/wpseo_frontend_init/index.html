wpseo_frontend_init

Used to load the required files on the plugins_loaded hook, instead of immediately.

Description

wpseo_frontend_init(); 

Usage

  1. if ( !function_exists( 'wpseo_frontend_init' ) ) { 
  2.     require_once ABSPATH . PLUGINDIR . 'yoast-seo/wp-seo-main.php'; 
  3. } 
  4.  
  5.   
  6. // NOTICE! Understand what this does before running. 
  7. $result = wpseo_frontend_init(); 
  8.     

Defined (1)

The function is defined in the following location(s).

/wp-seo-main.php  
  1. function wpseo_frontend_init() { 
  2.     add_action( 'init', 'initialize_wpseo_front' ); 
  3.  
  4.     $options = WPSEO_Options::get_option( 'wpseo_internallinks' ); 
  5.     if ( $options['breadcrumbs-enable'] === true ) { 
  6.         /** 
  7.          * If breadcrumbs are active (which they supposedly are if the users has enabled this settings,  
  8.          * there's no reason to have bbPress breadcrumbs as well. 
  9.          * 
  10.          * @internal The class itself is only loaded when the template tag is encountered via 
  11.          * the template tag function in the wpseo-functions.php file 
  12.          */ 
  13.         add_filter( 'bbp_get_breadcrumb', '__return_false' ); 
  14.     } 
  15.  
  16.     add_action( 'template_redirect', 'wpseo_frontend_head_init', 999 ); 
  17. }
