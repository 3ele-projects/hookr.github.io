  141. /** 
  142.  * Register a new variable replacement 
  143.  * 
  144.  * This function is for use by other plugins/themes to easily add their own additional variables to replace. 
  145.  * This function should be called from a function on the 'wpseo_register_extra_replacements' action hook. 
  146.  * The use of this function is preferred over the older 'wpseo_replacements' filter as a way to add new replacements. 
  147.  * The 'wpseo_replacements' filter should still be used to adjust standard WPSEO replacement values. 
  148.  * The function can not be used to replace standard WPSEO replacement value functions and will thrown a warning 
  149.  * if you accidently try. 
  150.  * To avoid conflicts with variables registered by WPSEO and other themes/plugins, try and make the 
  151.  * name of your variable unique. Variable names also can not start with "%%cf_" or "%%ct_" as these are reserved 
  152.  * for the standard WPSEO variable variables 'cf_<custom-field-name>', 'ct_<custom-tax-name>' and 
  153.  * 'ct_desc_<custom-tax-name>'. 
  154.  * The replacement function will be passed the undelimited name (i.e. stripped of the %%) of the variable 
  155.  * to replace in case you need it. 
  156.  * 
  157.  * Example code: 
  158.  * <code> 
  159.  * <?php 
  160.  * function retrieve_var1_replacement( $var1 ) { 
  161.  *        return 'your replacement value'; 
  162.  * } 
  163.  * 
  164.  * function register_my_plugin_extra_replacements() { 
  165.  *        wpseo_register_var_replacement( '%%myvar1%%', 'retrieve_var1_replacement', 'advanced', 'this is a help text for myvar1' ); 
  166.  *        wpseo_register_var_replacement( 'myvar2', array( 'class', 'method_name' ), 'basic', 'this is a help text for myvar2' ); 
  167.  * } 
  168.  * add_action( 'wpseo_register_extra_replacements', 'register_my_plugin_extra_replacements' ); 
  169.  * ?> 
  170.  * </code> 
  171.  * 
  172.  * @since 1.5.4 
  173.  * 
  174.  * @param  string $var              The name of the variable to replace, i.e. '%%var%%' 
  175.  *                                  - the surrounding %% are optional, name can only contain [A-Za-z0-9_-]. 
  176.  * @param  mixed  $replace_function Function or method to call to retrieve the replacement value for the variable 
  177.  *                                  Uses the same format as add_filter/add_action function parameter and 
  178.  *                                  should *return* the replacement value. DON'T echo it. 
  179.  * @param  string $type             Type of variable: 'basic' or 'advanced', defaults to 'advanced'. 
  180.  * @param  string $help_text        Help text to be added to the help tab for this variable. 
  181.  * 
  182.  * @return bool  Whether the replacement function was succesfully registered 
  183.  */ 
  184. function wpseo_register_var_replacement( $var, $replace_function, $type = 'advanced', $help_text = '' ) {
