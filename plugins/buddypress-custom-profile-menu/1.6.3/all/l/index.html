BuddyPress Custom Profile Menu

Version 1.6.3 – Finally a plugin that enables you to easily add tabs to the BuddyPress user profile. Just start by creating a normal menu under Appearance->Menus, then select this menu in the plugin’s settings page. Once you do that, your front-end BuddyPress profile menu will automatically be customized to fully reflect your new configuration. Enjoy :) By Sensible Plugins.
