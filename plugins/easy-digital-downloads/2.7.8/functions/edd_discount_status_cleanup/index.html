edd_discount_status_cleanup

Updates discounts that are expired or at max use (that are not already marked as so) as inactive or expired.

Description

edd_discount_status_cleanup(); 

Usage

  1. if ( !function_exists( 'edd_discount_status_cleanup' ) ) { 
  2.     require_once ABSPATH . PLUGINDIR . 'easy-digital-downloads/includes/discount-functions.php'; 
  3. } 
  4.  
  5.   
  6. // NOTICE! Understand what this does before running. 
  7. $result = edd_discount_status_cleanup(); 
  8.
