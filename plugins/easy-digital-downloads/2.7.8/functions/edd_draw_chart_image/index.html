edd_draw_chart_image

Draws Chart for PDF Report.

Description

(string) edd_draw_chart_image(); 

Draws the sales and earnings chart for the PDF report and then retrieves the URL of that chart to display on the PDF Report

Returns (string)

$chart->getUrl() URL for the Google Chart

Usage

  1. if ( !function_exists( 'edd_draw_chart_image' ) ) { 
  2.     require_once ABSPATH . PLUGINDIR . 'easy-digital-downloads/includes/admin/reporting/pdf-reports.php'; 
  3. } 
  4.  
  5.   
  6. // NOTICE! Understand what this does before running. 
  7. $result = edd_draw_chart_image(); 
  8.
