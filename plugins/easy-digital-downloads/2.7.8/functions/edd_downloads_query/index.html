edd_downloads_query

Downloads Shortcode.

Description

(string) edd_downloads_query( (array) $atts, (null) $content = null ); 

This shortcodes uses the WordPress Query API to get downloads with the arguments specified when using the shortcode. A list of the arguments can be found from the EDD Dccumentation. The shortcode will take all the parameters and display the downloads queried in a valid HTML

tags.

Returns (string)

$display Output generated from the downloads queried

Parameters (2)

0. $atts (array)
Shortcode attributes
1. $content — Optional. (null) => null
The content.
