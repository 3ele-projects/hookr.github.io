acf/create_field

The Advanced Custom Fields acf create field hook.

Description

do_action( 'acf/create_field', (callback) $array ); 

Parameters (1)

0. $array — Optional. (callback) => array( 'type' => 'number', 'name' => 'menu_order', 'value' => $post->menu_order, )
The array.

Usage

To run the hook, copy the example below.
  1. // run the action 
  2. do_action( 'acf/create_field', $array ); 
The following example is for adding a hook callback.
  1. // define the acf/create_field callback 
  2. function action_acf_create_field( $array ) { 
  3.     // make action magic happen here... 
  4. }; 
  5.          
  6. // add the action 
  7. add_action( 'acf/create_field', 'action_acf_create_field', 10, 1 ); 
To remove a hook callback, use the example below.
  1. // remove the action 
  2. remove_action( 'acf/create_field', 'action_acf_create_field', 10, 1 );
