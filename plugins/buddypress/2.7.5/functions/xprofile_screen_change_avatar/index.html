xprofile_screen_change_avatar

Handles the uploading and cropping of a user avatar.

Description

xprofile_screen_change_avatar(); 

Displays the change avatar page.

Usage

  1. if ( !function_exists( 'xprofile_screen_change_avatar' ) ) {

  4.
