bp_update_meta_cache

Update the metadata cache for the specified objects.

Description

(array|bool) bp_update_meta_cache( (array) $args = array() ); 

Based on WordPress's , this function primes the cache with metadata related to a set of objects. This is typically done when querying for a loop of objects; pre-fetching metadata for each queried object can lead to dramatic performance improvements when using metadata in the context of template loops.

Returns (array|bool)

Metadata cache for the specified objects, or false on failure.
