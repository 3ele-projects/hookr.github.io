bp_core_activate_signup

Activate a signup, as identified by an activation key.

Description

(int|bool) bp_core_activate_signup( (string) $key ); 

Returns (int|bool)

User ID on success, false on failure.

Parameters (1)

0. $key (string)
Activation key.
