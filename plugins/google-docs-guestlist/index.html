Google Docs Guestlist

Version 1.2 – A wedding guestlist that uses Google Docs for its backend. Instructions: Create a google docs spreadsheet with the following 7 headers: Guest Name, Code, Custom Message for Guest, Ceremony, Banquet, Message from Guest, Hotel. Go to “Settings->Google Docs Guestlist” to configure. Add the text: wpgc-googledocsguestlist in the content of your RSVP page. By Gifford Cheung, Brian Watanabe.
