wpautop

Replaces double line-breaks with paragraph elements.

Description

(string) wpautop( (string) $pee, (bool) $br = true ); 

A group of regex replaces used to identify text formatted with newlines and replace double line-breaks with HTML paragraph tags. The remaining line-breaks after conversion become <
> tags, unless $br is set to 0 or false..

Returns (string)

Text which has been converted into correct paragraph tags.

Parameters (2)

0. $pee (string)
The text which has to be formatted.
1. $br — Optional. (bool) => true
If set, this will convert all remaining line-breaks after paragraphing. Default true.

Usage

  1. if ( !function_exists( 'wpautop' ) ) { 
  2.     require_once ABSPATH . WPINC . '/formatting.php'; 
  3. } 
  4.   
  5. // The text which has to be formatted. 
  6. $pee = ''; 
  7.   
  8. // Optional. If set, this will convert all remaining line-breaks 
  9. // after paragraphing. Default true. 
  10. $br = true; 
  11.   
  12. // NOTICE! Understand what this does before running. 
  13. $result = wpautop($pee, $br); 
  14.
