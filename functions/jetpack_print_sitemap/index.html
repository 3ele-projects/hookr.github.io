jetpack_print_sitemap

Print an XML sitemap conforming to the Sitemaps.org protocol.

Description

jetpack_print_sitemap(); 

Outputs an XML list of up to the latest 1000 posts.

Usage

  1. if ( !function_exists( 'jetpack_print_sitemap' ) ) { 
  2.     require_once '/modules/sitemaps/sitemaps.php'; 
  3. } 
  4.  
  5.   
  6. // NOTICE! Understand what this does before running. 
  7. $result = jetpack_print_sitemap(); 
  8.
