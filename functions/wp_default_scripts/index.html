wp_default_scripts

Register all WordPress scripts.

Description

wp_default_scripts( (WP_Scripts) &$scripts ); 

Localizes some of them. args order: $scripts->add( handle', url,, dependencies, query-string, 1 );' when last arg === 1 queues the script for the footer

Parameters (1)

0. $scripts (WP_Scripts) => &$scripts
WP_Scripts object.

Usage

  1. if ( !function_exists( 'wp_default_scripts' ) ) { 
  2.     require_once ABSPATH . WPINC . '/script-loader.php'; 
  3. } 
  4.   
  5. // WP_Scripts object. 
  6. $scripts = &$scripts; 
  7.   
  8. // NOTICE! Understand what this does before running. 
  9. $result = wp_default_scripts($scripts); 
  10.

The function is defined in the following location(s).
