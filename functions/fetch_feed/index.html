fetch_feed

Build SimplePie object based on RSS or Atom feed from URL.

Description

(WP_Error|SimplePie) fetch_feed( (mixed) $url ); 

Returns (WP_Error|SimplePie)

WP_Error object on failure or SimplePie object on success
