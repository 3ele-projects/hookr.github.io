random_int

Fetch a random integer between $min and $max inclusive.

Description

(int) random_int( (int) $min, (int) $max ); 

Returns (int)

Parameters (2)

0. $min (int)
The min.
1. $max (int)
The max.

  8.      * ~PHP_INT_MAX or PHP_INT_MAX, we let it fail as not an integer. Floats

  26.  );

  32.      * integers along a given range. 
  33.      */

  36.             'Minimum value must be less than or equal to the maximum value' 
  37.  );

  42.     }

  76.          */

  78.         $mask = ~0; 
  79.

  83.          * $bits is effectively ceil(log($range, 2)) without dealing with  
  84.          * type juggling

  87.             if ($bits % 8 === 0) { 
  88.                ++$bytes; 
  89.             }

  119.  );
