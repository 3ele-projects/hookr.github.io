get_theme_starter_content

Expand a theme's starter content configuration using core-provided data.

Description

(array) get_theme_starter_content(); 

Returns (array)

Array of starter content.

Usage

  1. if ( !function_exists( 'get_theme_starter_content' ) ) { 
  2.     require_once ABSPATH . WPINC . '/theme.php'; 
  3. } 
  4.  
  5.   
  6. // NOTICE! Understand what this does before running. 
  7. $result = get_theme_starter_content();
