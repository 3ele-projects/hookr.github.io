wp_footer

The WordPress Core wp footer hook.

Description

add_action( 'wp_footer', (string) $wp_underscore_playlist_templates, (int) $int ); 

Parameters (2)

0. $wp_underscore_playlist_templates (string) => 'wp_underscore_playlist_templates'
The wp underscore playlist templates.
1. $int (int)
The int.

Usage

To run the hook, copy the example below.
  1. // run the action 
  2. do_action( 'wp_footer', $wp_underscore_playlist_templates, $int ); 
The following example is for adding a hook callback.
  1. // define the wp_footer callback 
  2. function action_wp_footer( $wp_underscore_playlist_templates, $int ) { 
  3.     // make action magic happen here... 
  4. }; 
  5.          
  6. // add the action 
  7. add_action( 'wp_footer', 'action_wp_footer', 10, 2 ); 
To remove a hook callback, use the example below.
  1. // remove the action 
  2. remove_action( 'wp_footer', 'action_wp_footer', 10, 2 );
