wp_image_editor

Loads the WP image-editing interface.

Description

wp_image_editor( (int) $post_id, (constant) $msg = false ); 

Parameters (2)

0. $post_id (int)
The post id.
1. $msg — Optional. (constant) => false
Message to display for image editor updates or errors. Default false.

Usage

  1. if ( !function_exists( 'wp_image_editor' ) ) { 
  2.     require_once ABSPATH . '/wp-admin/includes/image-edit.php'; 
  3. } 
  4.   
  5. // The post id. 
  6. $post_id = -1; 
  7.   
  8. // Optional. Message to display for image editor updates or errors. 
  9. // Default false. 
  10. $msg = false; 
  11.   
  12. // NOTICE! Understand what this does before running. 
  13. $result = wp_image_editor($post_id, $msg); 
  14.
