bizway_image_resize

This function resizes images It takes image source, width height and quality as a parameter This function is based on the approach described by Victor Teixeira here: http://core.trac.wordpress.org/ticket/15311.

Description

(array) bizway_image_resize( $img_url, $width, $height, (constant) $crop = true, (int) $quality = 100 ); 

Returns (array)

with image URL, width and height

Parameters (5)

0. $img_url
The img url.
1. $width
The width.
2. $height
The height.
3. $crop — Optional. (constant) => true
The crop.
4. $quality — Optional. (int) => 100
The quality.
