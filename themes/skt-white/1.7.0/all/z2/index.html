SKT White By SKT Themes

Version 1.7.0 – SKT White is a free responsive premium WordPress theme which is flexible, simple and adaptable for any type of website you want to create. So corporate, business, photography, wedding, industrial, restaurant or any sort of industry. SKT White has the ability to change the link colors. So make it suit your style you want. Comprises of easy to use sections on the homepage which make it super easy for your to manage the content on your homepage. Theme options are also one of the most easy to use and you can edit slider, and the homepage content from options. 100% responsive to the last pixel this theme is retina ready and hd ready and has ability to easily edit any part you want. Comes with default parallax slider.Default page templates, search and other standard pages readily available with this theme. Compatible with popular plugins like WooCommerce making it ecommerce/shop ready and Contact form 7.It is translation ready and can be translated for any language. Has custom header and custom background. Check out the demo at: http://sktthemesdemo.net/skt-white-pro/
